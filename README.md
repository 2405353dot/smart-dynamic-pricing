\# Smart Dynamic Pricing System



\## What is this project?

This project helps decide the best product price using Machine Learning.



It predicts:

\- demand

\- profit

\- best price



\## What does it do?

\- You enter your price

\- You enter competitor price

\- It tells:

&#x20; - expected demand

&#x20; - expected profit

&#x20; - best price to choose



\## Technologies used

\- Python

\- Machine Learning (Random Forest)

\- Streamlit (for UI)



\## How it works

1\. Data is generated

2\. Model learns demand

3\. Profit is calculated

4\. Best price is selected



\## Why this project is useful

This can help companies:

\- set better prices

\- increase profit

\- understand competition



\## Output

\- Demand prediction

\- Profit calculation

\- Best price suggestion

\- Graph

\- Competitor analysis table
